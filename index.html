Paulina Gómez de la Casa Rodríguez
Fit to Scrape

Aplicación web que permite a los usuarios ver y dejar comentarios sobre las últimas noticias. Usando Mongoose y Cheerio, esta aplicación obtiene las noticias desde otro sitio.

Fit Connect Web App

La herramienta Business Web App, permite al usuario tener la administración adecuada de su negocio. Diseño personalizado para un negocio de franquicia de gimnasio.

Mythology Game

Juego interactivo para navegadores web con temático de la mitología griega. La aplicación actualiza dinámicamente las páginas HTML con la biblioteca jQuery. Selecciona tu jugador para comenzar la batalla de los dioses.


Bamazon

Aplicación de tienda tipo Amazon con base de datos MySQL. La aplicación recibirá pedidos de los clientes y actualizara el inventario de la tienda.

WL Plan

Herramienta de aplicación web, para usuarios que quieren perder peso pero que no cuentan con tiempo o presupuesto para asistir a un nutriólogo. El usuario puede ingresar información personal y se mostrará un plan de recibo con gráficos sobre su peso real.

Trivia Game

Juego de trivia usando JavaScript para la lógica y jQuery para manipular HTML. El usuario tiene que responder antes de que se acabe el tiempo, Disney Theme Trivia. ¡Que te diviertas!
